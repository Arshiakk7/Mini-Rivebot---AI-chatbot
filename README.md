# Mini Rivebot - AI chatbot 

**Overview**
- Mini RiveBot is a simple AI chatbot built using Python, Tkinter, and RiveScript.
- It features a clean GUI, chat history saving, and customizable chatbot responses.

**Features**
 - Interactive chatbot using RiveScript
 - GUI built with Tkinter
 - Clean UI (light green background + white chat area)
 - Chat history saved to file
 - Press Enter to send message
 - Clear chat button
 - Type "quit" to exit

**Technology used**
- Python
- Tkinter
- Rivescript

**Project structure**
 - simplechatbot
 - brain.rive
 - chat_history.txt
 - chatbot.py

  **how to run**
 - install Rivescript
 -  pip install rivescript

  **Run the program**
  - python chatbot.py

  **Customize bot**
- Edit brain.rive file to add your own response
